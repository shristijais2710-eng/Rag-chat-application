# RAG Chat Application

A smart AI-powered chatbot built using Python and FastAPI that uses Retrieval-Augmented Generation (RAG) to provide accurate and context-based responses from a custom Python knowledge base.

## Features

* AI-powered chatbot
* RAG architecture implementation
* Python-based knowledge base
* FastAPI backend
* Modern UI using HTML, CSS, and JavaScript
* Context-aware response generation

## Tech Stack

* Python
* FastAPI
* HTML
* CSS
* JavaScript
* AI / RAG Architecture

## Project Overview

This project allows users to ask questions related to Python. The chatbot retrieves relevant information from the provided knowledge base and generates intelligent responses in real time.

## Learning Outcomes

* API development with FastAPI
* Frontend and backend integration
* RAG architecture implementation
* AI response handling
* Building intelligent chatbot systems
